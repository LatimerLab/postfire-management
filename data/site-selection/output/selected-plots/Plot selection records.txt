## plot naming

# I F C Y S ER ER P

# I = iteration of plot set
# F = fire
# C = plot category
# Y = planting year
# S = seed tree distance
# ER = elev and rad quad
# ER = elev and rad subquad
# P = priority (which tier)
# T = type (Trt, control, interior)


For C
1 is perimeter plot, no management, restricted env range
2 is perimeter plot, no management, broader env range
3 is internal plot, no management, broader env. range
